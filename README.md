# UChat
> a chat room based on nodeJS

### 启动步骤

1. 克隆项目到本地，命令行进入UChat, `cd UChat`;
2. 安装相应的依赖， `npm install`;
3. 启动服务，`npm start`;
4. 在浏览器地址栏输入`localhost:7777`查看效果
